# Possible Errors

| ErrorCode | Explanation |
| --------- | ----------- |
| -10002 to -10004 | There is no valid license for Barcode Decodeing. |
| -10030 to -10059 | Failed to parse the template file `template_cvr.json`. |
| -70060 | There is no valid license for Batch Scanning. |
| -70063 | Failed to parse the template file `template_panorama.json`. |
| -80001 | Failed to parse the template file `template_ui.json`. |
