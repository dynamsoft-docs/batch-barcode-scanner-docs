## Color Reference

| Color | Description |
| --------- | ----------- |
| Green | Recognized barcodes |
| Light blue | Newly recognized barcodes |
| Orange | Unrecognized barcodes by layout analysis or barcode location detection|
| Purple | Wanted barcodes |
